# CityPulse

CityPulse is a civic issue reporting and city management dashboard. Citizens can
submit and track complaints, while administrators can monitor services,
emergencies, traffic, utilities, and reports.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Local Development

Requirements: Node.js and npm.

```sh
npm install
npm run dev
```

The development server runs at `http://localhost:8080`.

## Available Scripts

```sh
npm run build
npm run lint
npm run test
npm run preview
```
